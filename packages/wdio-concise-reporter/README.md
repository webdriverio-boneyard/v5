WDIO Concise Reporter
=====================

> A WebdriverIO plugin to report in concise style.

## Configuration

The following code shows the default wdio test runner configuration. Just add `'concise'` as a reporter
to the array.

```js
// wdio.conf.js
module.exports = {
  // ...
  reporters: ['dot', 'concise'],
  // ...
};
```
